# Private Diary App

A privacy focused journal app.

<kbd>
  <img src="docs/diary-app.webp?raw=true"> 
</kbd>

## Features

- Offline first
- Sync to Google Drive (Optional)
- Privacy
- Password protection

## Dependencies

- React Native
- Mobx-state-tree
- Realm
- UI Kitten
- React Navigation v6

## Installation

Clone this repo

```
git clone https://github.com/saheeranas/diaryapp.git
```

Open project folder and install dependencies

```
cd diaryapp
yarn
or
npm i
```

Run the project

```
yarn android
or
npm run android
```

or

```
yarn ios
or
npm run ios
```
